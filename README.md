# Flight_booking_app_Webflux_MongoDB

## Project Goals
- Build reactive Flight Ticket Booking system using Spring WebFlux + MongoDB (reactive)
- - Implement REST APIs for search, inventory add, booking, ticket retrieval, booking history and cancellation.
- Support user flows: search, book, view by PNR, history by email, cancel (>=24h before)
- Admin: add inventory/schedules

## Tech Stack
- Java 17+
- Spring Boot (WebFlux)
- Spring Data Reactive MongoDB
- Maven
- JUnit5 + WebTestClient
